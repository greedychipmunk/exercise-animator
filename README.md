# Exercise Animator

Demonstrate any exercise with AI-generated animations! This Space uses the **Wan2.1** model via API to generate high-quality, instructional videos of human movement.

## How to use
1. Enter the name of an exercise (e.g., "Jumping Jacks").
2. Click "Animate".
3. Wait a few moments for the AI to generate the video.
